# GTA V QA Testing Project

## 📌 Project Overview

This project demonstrates manual testing performed on GTA V, focusing on identifying gameplay bugs, documenting issues, and designing structured test cases.

## 🛠 Tools Used

* Jira (Bug Tracking)
* Excel (Test Case Design)
* GTA V (Test Environment)

## 🐞 Bug Reports

Bugs were reported in Jira and documented here with:

* Summary
* Steps to Reproduce
* Expected vs Actual Result
* Severity & Reproducibility

## 📊 Test Cases

Includes structured test cases covering:

* Gameplay mechanics
* NPC behavior
* Vehicles
* Environment interactions

## 🎥 Evidence

Screenshots and videos attached for reproducibility and validation.

## 🎯 Skills Demonstrated

* Manual Testing
* Bug Reporting
* Test Case Design
* QA Documentation
